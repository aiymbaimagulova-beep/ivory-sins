# Ivory Sins

React clothing e-commerce demo with checkout, favorites, lookbook и авторизацией.

## Как запустить локально

1. Перейдите в папку проекта:

```bash
cd "c:\\Users\\Asus\\Desktop\\ivory sinsss aiym\\ivory-sins"
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите сервер API:

```bash
npm run server
```

4. Запустите фронтенд React:

```bash
npm start
```

5. Откройте ссылки в браузере:

- http://localhost:3000 (фронтенд)
- http://localhost:5000/api/status (API)

## Команды

- `npm start` — запуск фронтенда в dev режиме.
- `npm run server` — запуск Express API.
- `npm run build` — production сборка.

## Стек технологий

- React, react-router-dom
- Express, sqlite3, bcryptjs
- CSS, локальное хранилище (localStorage)

## API

- `POST /api/register` (email, password)
- `POST /api/login` (email, password)
- `GET /api/status` (статус сервера)

## Дополнительно

- Автоматическая тема (dark/light + localStorage)
- Базовая валидация формы (email, непустые поля)
- 404-страница, обработка ошибок
